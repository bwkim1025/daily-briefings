## PAPERS
### 철선 없이 '조영영상'만으로 협착을 판단하다 — FAST III
압력철선·충혈제 없이 3D 정량 조영영상에서 도출한 vFFR이 표준 FFR에 비열등함을 입증했습니다.
#### 어떤 연구
순환기(중재시술). 만성관상동맥증후군·불안정협심증·NSTEMI 환자의 중등도 협착(30~80%) 2,235명을 vFFR 유도군 vs FFR 유도군으로 무작위 배정한 다기관 공개 3상.
#### 핵심 결과
1년 복합사건(사망·심근경색·재관류)은 vFFR 7.5% vs FFR 7.5%로 비열등(비열등성 P=0.004). 전사망(2.2% vs 2.3%)·심근경색(2.9% vs 2.4%) 차이 없음. 시술시간은 55.8분 vs 60.9분으로 단축, 조영제·방사선 부담도 감소.
#### 임상 적용
별도 철선·충혈제 없이 생리학적 평가가 가능해지면, 한국 도입 시 시술 효율과 환자 안전(조영제·방사선) 측면에서 이점이 큽니다. 다만 영상 품질에 의존하므로 표준화·교육이 전제입니다.
#### 인용 정보
- 인용수: 정량 인용수 미검증(자동 실행 환경, 인용 API 미접근). 선정 근거: 임상 영향 + ACC.26 동시발표·NEJM 게재(검색일 2026-06-18)
- 발표일: 2026-03-29 (ACC.26, 최근 6개월 이내 확인)
- 저널: N Engl J Med 2026. (FAST III)
- DOI: 10.1056/NEJMoa2601841

### 대장암 수술 후 '운동 처방'이 재발·사망을 줄인다 — CHALLENGE
보조항암화학요법을 마친 대장암 환자에게 3년 구조화 운동 프로그램이 무병생존·전체생존을 유의하게 개선했습니다.
#### 어떤 연구
종양내과(대장암). 보조항암 완료 환자 889명을 3년 구조화 운동군 vs 건강교육군으로 무작위 배정한 국제 다기관 3상(CCTG CO.21), 중앙 추적 7.9년.
#### 핵심 결과
무병생존 위험 28% 감소(HR 0.72), 5년 무병생존 80.3% vs 73.9%. 전체생존 위험 37% 감소(HR 0.63). ASCO 2026에서는 비용효용 분석으로 '치료 표준 권고' 가치가 재확인됐습니다.
#### 임상 적용
'운동은 약이다'를 근거 수준에서 입증한 시험으로, 한국에서도 대장암 보조치료 종료 환자에게 구조화된 운동 상담·연계를 표준 권고로 제시할 근거가 됩니다.
#### 인용 정보
- 인용수: 정량 인용수 미검증(자동 실행 환경, 인용 API 미접근). 선정 근거: 임상 영향(랜드마크) + ASCO 2026 비용효용 재확인(검색일 2026-06-18)
- 발표일: 2025-06-01 (NEJM) — ※ 6개월 윈도우 밖, 백필 선정 사유는 편집자 노트 참조
- 저널: N Engl J Med 2025. (CHALLENGE, CCTG CO.21)
- DOI: 10.1056/NEJMoa2502760
